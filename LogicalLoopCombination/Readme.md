# Level 5: Logical Loop Combinations 🚀

This repository contains Java programs solving advanced logical problems using loops and conditional logic.  
Each problem is designed to strengthen understanding of number properties, digit manipulation, and iterative constructs.

---

## Problems Included

1. **Sum of Digits Even**  
   Print all numbers between 1–100 whose sum of digits is even.

2. **Divisible by 7 but not 5**  
   Count how many numbers between 1–500 are divisible by 7 but not by 5.

3. **Palindrome Numbers**  
   Print all numbers that are palindromes between 1–500.

4. **Digits Sum Multiple of 3**  
   Print numbers between 1–100 whose digits add up to a multiple of 3.

5. **Smallest and Largest Digit**  
   Find the smallest and largest digit in a given number.

6. **Binary Representation with Even 1s**  
   Print all numbers from 1–n whose binary representation has an even number of 1s.

7. **Pattern Printing (i × i)**  
   Print a pattern where each row `i` prints `i*i`.

8. **Factorial Series**  
   Print factorial of each number from 1 to n.

9. **Sum of Odd and Even Digits**  
   Print the sum of all odd digits and even digits separately in a number.

10. **Skip Zero with Continue**  
    Take 5 numbers as input. If the user enters 0, skip it using `continue`.  
    At the end, print the sum of all non-zero numbers entered.
