# Array Problems in Java

This repository contains Java programs that demonstrate fundamental operations on arrays.  
Each problem focuses on building logic and practicing array manipulation techniques.

---

## 📋 Problem Set

1. **Input and Print Array**  
   - Take `n` integers as input into an array.  
   - Print all elements of the array.

2. **Sum of Elements**  
   - Find and print the sum of all elements in the array.

3. **Average of Elements**  
   - Calculate and print the average of array elements.

4. **Maximum Element**  
   - Find and print the maximum element in the array.

5. **Minimum Element**  
   - Find and print the minimum element in the array.

6. **Count Positive, Negative, and Zero Elements**  
   - Count how many elements are positive, negative, or zero.

7. **Count Even and Odd Elements**  
   - Count how many elements are even and how many are odd.

8. **Index of Maximum Element**  
   - Find and print the index of the maximum element in the array.

9. **Index of Minimum Element**  
   - Find and print the index of the minimum element in the array.
