# Level 4: Logical Operators & Compound Statements ⚡

This repository contains Java programs that demonstrate the use of **logical operators** and **compound statements**.  
Each problem is implemented as a standalone program for clarity and practice.

---

## 📂 Problems Included

1. **Character Type Check**  
   - Input: A single character  
   - Output: Whether it is a letter, digit, or neither  

2. **FizzBuzz**  
   - Input: A number  
   - Output:  
     - "Fizz" if divisible by 3  
     - "Buzz" if divisible by 5  
     - "FizzBuzz" if divisible by both  

3. **Median of Three Numbers**  
   - Input: Three numbers  
   - Output: The median value (neither maximum nor minimum)  

4. **24-Hour Time → AM/PM**  
   - Input: Hours and minutes (24-hour format)  
   - Output: Whether it is AM or PM  

5. **Tax Eligibility**  
   - Input: Age and income  
   - Output: Eligible if age > 18 and income > 5 Lakh  

6. **Positive Sum Check**  
   - Input: Two numbers  
   - Output: Whether both are positive and their sum < 100  

7. **Digit to Word (0–9)**  
   - Input: A single digit  
   - Output: Word form ("Zero" to "Nine")  

8. **Weekday or Weekend**  
   - Input: Weekday number (1–7)  
   - Output: Whether it is a weekday or weekend  

9. **Electricity Bill Calculation (Slabs)**  
   - Input: Units consumed  
   - Output: Bill amount calculated using if-else slabs  

10. **Password Validation**  
    - Input: A password string  
    - Output: Valid if length ≥ 8 and contains at least one digit
