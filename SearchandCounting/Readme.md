# Level 2: Searching & Counting Logic (Java)

This repository contains Java programs that focus on searching and counting operations in arrays.  
These problems help strengthen logical thinking and array manipulation skills.

---

## 📋 Problem Set

1. **Search Element**  
   - Input an element `x` and check if it exists in the array.

2. **Count Occurrences**  
   - Count how many times a given element appears in the array.

3. **First Occurrence**  
   - Find the index of the first occurrence of a given number.

4. **Last Occurrence**  
   - Find the index of the last occurrence of a given number.

5. **Check Uniqueness**  
   - Verify if all elements in the array are unique.

6. **Sum of Even Elements**  
   - Calculate the sum of even elements only.

7. **Sum of Odd Elements**  
   - Calculate the sum of odd elements only.

8. **Count Prime Numbers**  
   - Count how many prime numbers exist in the array.

9. **Divisible by 3 and 5**  
   - Count how many numbers are divisible by both 3 and 5.

10. **Count Perfect Squares**  
    - Count how many elements in the array are perfect squares.
