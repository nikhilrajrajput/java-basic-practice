# Level 2: Number-based Looping Logic 🚀

This repository contains Java programs that demonstrate number-based looping logic.  
Each problem focuses on iterative constructs, digit manipulation, and mathematical checks.

---

## 📘 Problem Set

1. **Count Digits**  
   - Input: An integer  
   - Output: Number of digits in the integer  

2. **Reverse a Number**  
   - Input: An integer  
   - Output: Reversed integer  

3. **Palindrome Check**  
   - Input: An integer  
   - Output: Whether the number is a palindrome  

4. **Sum of Digits**  
   - Input: An integer  
   - Output: Sum of all digits  

5. **Armstrong Number Check**  
   - Input: An integer  
   - Output: Whether the number is an Armstrong number  
   - Example: 153 → 1³ + 5³ + 3³ = 153 ✅

6. **Perfect Number Check**  
   - Input: An integer  
   - Output: Whether the number is a perfect number  
   - Example: 28 → 1 + 2 + 4 + 7 + 14 = 28 ✅

7. **Prime Numbers (1–100)**  
   - Output: All prime numbers between 1 and 100  

8. **Prime Check**  
   - Input: An integer  
   - Output: Whether the number is prime  

9. **Fibonacci Series**  
   - Input: n terms  
   - Output: Fibonacci sequence up to n terms  

10. **Sum of Fibonacci Terms**  
    - Input: n terms  
    - Output: Sum of first n Fibonacci numbers
