# Java Programming Exercises – Loops & Number Logic

This repository contains beginner-friendly Java programs that demonstrate the use of **loops, conditionals, and basic number logic**. Each problem is designed to strengthen understanding of iteration, arithmetic operations, and input handling.

---

## 📘 Problem Set

1. **Print numbers from 1 to 10**  
   - Use a `for` loop to display numbers sequentially.

2. **Print all even numbers between 1 and 100**  
   - Iterate through numbers and check divisibility by 2.

3. **Print all odd numbers between 1 and 100**  
   - Iterate through numbers and check divisibility by 2.

4. **Print numbers from 10 down to 1**  
   - Use a decrementing `for` loop.

5. **Print the table of a given number (n × 1 to n × 10)**  
   - Input a number `n` and display its multiplication table.

6. **Print the sum of first n natural numbers**  
   - Formula: `sum = n * (n + 1) / 2`.

7. **Print the sum of all even numbers up to n**  
   - Loop through numbers up to `n` and add those divisible by 2.

8. **Print the sum of all odd numbers up to n**  
   - Loop through numbers up to `n` and add those not divisible by 2.

9. **Print the factorial of a given number**  
   - Use a loop or recursion to compute factorial.

10. **Print the product of digits of a given number**  
    - Extract digits using `%` and `/` operators, multiply them together.
