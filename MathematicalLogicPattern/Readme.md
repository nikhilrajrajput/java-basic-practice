# Number Logic & Series Problems (Java)

This repository contains Java programs solving classic number logic and series problems using loops and conditional logic.  
Each problem is designed to strengthen understanding of iteration, divisibility, and mathematical sequences.

---

## Problems Included

1. **Squares of Numbers**  
   Print the squares of numbers from 1 to `n`.

2. **Cubes of Numbers**  
   Print the cubes of numbers from 1 to `n`.

3. **Divisible by 7**  
   Print all numbers between `a` and `b` that are divisible by 7.

4. **HCF (GCD) of Two Numbers**  
   Find the highest common factor (greatest common divisor) of two numbers using loops.

5. **LCM of Two Numbers**  
   Find the least common multiple of two numbers using loops.

6. **Factors of a Number**  
   Print all factors of a given number.

7. **Sum of Factors**  
   Find the sum of all factors of a given number.

8. **Strong Number Check**  
   Check if a number is a *strong number* (sum of factorials of digits equals the number itself).

9. **Arithmetic Progression (AP)**  
   Print the first `n` terms of an arithmetic progression given first term `a` and common difference `d`.

10. **Geometric Progression (GP)**  
    Print the first `n` terms of a geometric progression given first term `a` and common ratio `r`.
