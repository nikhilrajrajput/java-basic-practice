# Level 3: Math and Number Logic 🚀

This repository contains Java programs that solve classic math and number logic problems.  
Each problem is implemented as a standalone program for clarity and practice.

---

## 📂 Problems Included

1. **Check Distinct Digits (3-digit number)**  
   - Input: A 3-digit number  
   - Output: Whether all digits are distinct  

2. **Middle Digit Comparison (3-digit number)**  
   - Input: A 3-digit number  
   - Output: Whether the middle digit is the largest, smallest, or neither  

3. **First and Last Digit Equality (4-digit number)**  
   - Input: A 4-digit number  
   - Output: Whether the first and last digits are equal  

4. **Digit Category Check**  
   - Input: Any integer  
   - Output: Whether it is single-digit, double-digit, or multi-digit  

5. **Multiple of 7 or Ends with 7**  
   - Input: Any integer  
   - Output: Whether it is divisible by 7 or ends with digit 7  

6. **Quadrant Finder (Coordinates)**  
   - Input: Coordinates `(x, y)`  
   - Output: Which quadrant the point lies in, or if it’s on an axis/origin  

7. **Currency Division**  
   - Input: An amount  
   - Output: Whether it can be evenly divided into 2000, 500, or 100 notes  

8. **Range Check [100–999]**  
   - Input: Any integer  
   - Output: Whether the number lies within the range `[100, 999]`  

9. **Third Angle of Triangle**  
   - Input: Two angles of a triangle  
   - Output: The third angle, with validity check  

10. **Perfect Square Check (No sqrt)**  
    - Input: Any integer  
    - Output: Whether the number is a perfect square (without using `Math.sqrt`)
