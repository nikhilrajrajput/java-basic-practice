# 🧩 Level 2: Nested If & Multiple Conditions

This repository contains practice problems focused on **nested if statements** and **multiple conditions** in programming.  
Each problem helps strengthen logical thinking and conditional branching skills.

---

## 📌 Problem Set

1. **Triangle Validation**
   - Take three sides and check if they form a valid triangle.
   - If valid, determine whether it is:
     - Equilateral
     - Isosceles
     - Scalene

2. **Grading System**
   - Input marks (0–100).
   - Print the corresponding grade:
     - A / B / C / D / F

3. **Multiple Check**
   - Check if one of two given numbers is a multiple of the other.

4. **Greeting by Time**
   - Take the hour of the day (0–23).
   - Print:
     - Good Morning
     - Good Afternoon
     - Good Evening
     - Good Night

5. **Voting Eligibility**
   - Input age.
   - Check if eligible to vote (18+).

6. **Even/Odd Check**
   - Take two numbers.
   - Determine:
     - Both even
     - Both odd
     - One even and one odd

7. **Alphabet Range**
   - Take an alphabet character.
   - Check if it lies between:
     - `a–m`
     - `n–z`

8. **Day of the Week**
   - Input day number (1–7).
   - Print the corresponding day name.

9. **Month Days**
   - Input month number (1–12).
   - Print the number of days in that month.
   - *(Ignore leap years)*
